# 💌 Love Note

A beautifully animated static website to write and share heartfelt love notes.

---

## ✨ Features

- 💫 Smooth, romantic animations and transitions
- 🎨 Clean and elegant UI built with HTML, CSS & JavaScript
- 📱 Fully responsive — works on all screen sizes

---

## 🛠️ Tech Stack

- **HTML5** — Structure
- **CSS3** — Styling & animations
- **JavaScript** — Interactivity

---

## 🚀 Setup Instructions

### 1. Clone the Repository

```bash
git clone https://github.com/yourusername/love-note.git
cd love-note
```

### 2. Open Locally

No build tools or dependencies needed. Simply open the `index.html` file in your browser:

```bash
# On macOS
open index.html

# On Windows
start index.html

# On Linux
xdg-open index.html
```

Or drag and drop `index.html` into any browser window.

---

## 🌐 Deploy on GitHub Pages

1. Push the project to a GitHub repository
2. Go to **Settings → Pages**
3. Under **Source**, select `main` branch and `/ (root)` folder
4. Click **Save**
5. Your site will be live at:

```
https://yourusername.github.io/love-note
```

---

## 📁 Project Structure

```
love-note/
├── index.html       # Main HTML file
├── style.css        # Styles & animations
├── script.js        # JavaScript logic
└── README.md        # Project documentation
```
